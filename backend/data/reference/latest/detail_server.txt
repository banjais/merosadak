2️⃣ Cloudflare Worker Caching

Cloudflare can cache API responses.

Example for your /api/roads:

export default {
  async fetch(request, env) {
    const cache = caches.default;

    // Only cache GET requests
    if (request.method === "GET") {
      const cachedResponse = await cache.match(request);
      if (cachedResponse) return cachedResponse;
    }

    // Try primary backend
    let res;
    try {
      res = await fetch('https://sadaksathi.web.app/api' + request.url.replace(self.location.origin, ''));
      if (!res.ok) throw new Error('Primary backend failed');
    } catch {
      // fallback to Render
      res = await fetch('https://sadaksathi.onrender.com' + request.url.replace(self.location.origin, ''));
    }

    // Cache GET response for 30s at edge
    if (request.method === "GET") {
      const responseClone = res.clone();
      res.headers.set("Cache-Control", "public, max-age=30");
      await cache.put(request, responseClone);
    }

    return res;
  }
};


✅ Benefits:

Cloudflare serves cached response immediately

Reduces load on Render / Firebase

Edge caching makes app fast globally
